# 🐦 Mini Twitter - Spring Security

Projeto desenvolvido para estudo de autenticação e autorização utilizando **Spring Security**.

A aplicação simula um mini Twitter, onde usuários podem se registrar, autenticar e interagir com postagens.

---

## 📚 Tecnologias utilizadas

- Java 17+
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- Maven
- JPA / Hibernate
- Banco de dados MySQL

---

## 🔐 Funcionalidades de segurança

- ✅ Autenticação com JWT
- ✅ Criptografia de senha com BCrypt
- ✅ Controle de acesso por roles
- ✅ Filtros de segurança customizados
- ✅ Stateless authentication

---

## ⚙️ Funcionalidades da aplicação

- 👤 Cadastro de usuários
- 🔐 Login com geração de token
- 📝 Criação de posts
- 📄 Listagem de posts
- 🔒 Rotas protegidas
